Syntax and Hello, World!

Python has a simple and readable syntax. As mentioned before, to print "Hello, World!" to the console, you simply need to use the print() function:

print("Hello, World!")

In Python, strings can be enclosed in either double or single quotes; both are valid.

Variables and Data Types

In Python, you don't need to explicitly declare the type of a variable; Python infers it based on the assigned value. Here are some examples of variables and their types:

# Numeric variables
age = 25 average_grade = 8.5

# Text (string) variables
name = "John" message = 'Hello, how are you?'

# Boolean variables
is_adult = True has_discount = False

Lists and Dictionaries

Python provides data structures like lists and dictionaries to store collections of elements.

  • a. Lists: Lists are ordered and mutable collections of elements:
    • fruits = ["apple", "orange", "banana"]
    numbers = [1, 2, 3, 4, 5]
    mixed = [10, "Python", True]


    # Accessing list elements:
    print(fruits[0]) # Output: "apple"


    # Adding elements to the list:
    fruits.append("grape")
    print(fruits) # Output: ["apple", "orange", "banana", "grape"]
  • b. Dictionaries: Dictionaries are unordered and mutable collections that store key-value pairs:
    • person = {
    "name": "John",
    "age": 30,
    "occupation": "engineer"
    }

    # Accessing dictionary values:
    print(person["name"]) # Output: "John"


    # Adding a new key-value pair:
    person["city"] = "New York"
    print(person) # Output: {"name": "John", "age": 30, "occupation": "engineer", "city": "New York"}
Functions

Functions are reusable blocks of code that perform specific tasks. Here's a simple example:

def sum_numbers(a, b):
return a + b


result = sum_numbers(5, 3)
print(result) # Output: 8
Modules and Libraries

Python has a wide range of modules and libraries that facilitate specific tasks.

The math library in Python is a built-in module that provides a variety of mathematical functions and constants. It is part of the Python Standard Library, which means you don't need to install any external packages to use it.

In the provided example, we used the sqrt() function from the math module to calculate the square root of 25. The math.sqrt() function takes a single argument (a number) and returns its square root as a floating-point number.

For example, the math library provides mathematical functions:

import math


square_root = math.sqrt(25)
print(square_root) # Output: 5.0


# You can refer to the documentation for more functions: https://docs.python.org/3/library/math.html
Object Oriented Programming

Python is an object-oriented language, allowing you to create classes and objects. Though this might be more advanced, here's a simple example:

class Person:
def __init__(self, name, age):
self.name = name
self.age = age


def greet(self):
print(f"Hello, my name is {self.name} and I am {self.age} years old.")


john = Person("John", 25)
john.greet() # Output: "Hello, my name is John and I am 25 years old."

Remember that practice is essential to learn programming, so don't hesitate to try, experiment, and have fun coding in Python!

Object-Oriented Programming (OOP) is a programming paradigm that focuses on creating objects that encapsulate data and behavior. These objects are instances of classes, which act as blueprints for creating objects with shared attributes and methods. Python is known for its strong support for OOP, making it easy to implement and organize complex systems and applications.
